eMorpho Data Structure: fpga_histogram

Summary

The fpga_histogram data structure contains the MCA energy histogram.

Data structure

The fpga_histogram structure simply contains an array of 4096 32-bit unsigned int32 (uint32), which comprise the histogram. No further named fields or user dictionaries are available.
