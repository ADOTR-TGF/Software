rad_config – User Configuration Data

Introduction

The rad_config folder contains JSON configuration files for the individual clients and servers that constitute the Morpho software.

For each software component there is a data folder containing the configuration data.

The rad_config folder essentially carries user data and should not be overwritten when a new version of the software is installed. To be on the safe side, it is prudent to have a backup copy of this folder before installing a software update.
